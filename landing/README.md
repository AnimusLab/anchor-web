# Anchor Landing — Bloomberg Terminal Aesthetic

This is the high-density public showcase for the Anchor Governance Engine. It uses a "Bloomberg Terminal" aesthetic to convey professional authority and data-rich real-time audit transparency.

## ✨ Key Features
- **Terminal Boot Sequence**: Custom "system boot" animation with kernel logs and system diagnostic messages.
- **Dynamic Data Visualization**: Real-time Sparklines and MiniBarCharts for compliance trends and rule-loading stats.
- **High-Density UI**: Judicial gold accents on a deep obsidian background, using premium typography (Playfair Display & DM Mono).

## 🛠 Tech Stack
- React + Vite
- CSS Keyframe Animations (Hardware Accelerated)
- SVG-based data visualization components

## 🚀 Development
```bash
npm install
npm run dev
```
