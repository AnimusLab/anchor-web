# Anchor Dashboard — User Control Plane

The primary interface for enterprise organizations to monitor their AI fleet compliance, view real-time audit logs, and manage regulatory frameworks.

## ✨ Key Features
- **Live Governance Stream**: Real-time WebSocket feed of audit events across the jurisdictional mesh.
- **Compliance Counters**: Visual roll-ups of active rules, domain coverage, and current violation count.
- **World Monitor**: High-level geographic and network-topographical view of compliance node health.

## 🛠 Tech Stack
- React + Router
- Framer Motion animations
- WebSocket (WSS) integration

## 🚀 Development
```bash
npm install
npm run dev
```
